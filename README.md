# POWERSHELL+

PowerShell+ is a lightweight, transparent terminal for Windows built entirely with pure Win32 — no .NET, no runtime, no bloat. It wraps PowerShell with a sleek frameless GUI featuring a custom-drawn titlebar, glowing close button, and 84% opacity so your desktop always shows through.
Commands run exactly like PowerShell, but the shell itself uses a fraction of the memory. History navigation with arrow keys, auto-clear every 7 commands, and a cyan-accented dark UI make it feel like a terminal from the future — without the weight of one.
Features:

Frameless, draggable, resizable transparent window
Custom close button with hover glow effect
Command history (↑/↓ arrow keys)
Auto-clears after every 7 commands
Built-in cd, history, help, cls commands
All other commands passed directly to PowerShell
~2–5MB RAM idle vs ~80–150MB for standard PowerShell
Single .exe, no installer, no dependencies

Windows 10+ required.
